┌─ CRYPTOGRAPHIC LAYER ──────────────────────────────────────────────────────────┐
│                                                                                 │
│  Encryption:   3DES (Triple Data Encryption Standard)                         │
│  ├── Key Size: 192 bits (24 bytes)                                            │
│  ├── Block Size: 64 bits (8 bytes)                                            │
│  ├── Mode: Cipher Block Chaining (CBC)                                        │
│  ├── IV: Random 64-bit per encryption                                         │
│  ├── Padding: PKCS7                                                            │
│  └── Library: PyCryptodome                                                     │
│                                                                                 │
│  Hashing:      SHA-256, SHA-1, MD5                                            │
│  ├── SHA-256: 256-bit output (primary)                                        │
│  ├── SHA-1: 160-bit output (legacy support)                                   │
│  ├── MD5: 128-bit output (legacy support)                                     │
│  └── Library: Python hashlib                                                   │
│                                                                                 │
│  Random Generation: Cryptographically Secure                                   │
│  ├── Source: PyCryptodome.Random.get_random_bytes()                           │
│  ├── Entropy: OS-provided randomness                                           │
│  └── Purpose: IV & Key Generation                                              │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
